Preguntas

- Internet es una red global de redes, mientras que la Web, también conocida como World Wide Web (www), es una recopilación de información a la que se accede a través de Internet.
- Internet es infraestructura mientras que la Web es un servicio sobre esa infraestructura.
- El Internet como una gran librería, mientras que la web se puede ver como una colección de libros en esa tienda.

Las peticiones al servidor suelen realizarse mediante HTTP utilizando el método de petición GET, en el que el recurso se solicita a través de la URL al servidor web.

Código 200: esto no es un error HTTP, sino todo lo contrario.

Significa que la página ha cargado de forma correcta. En general todos los códigos de estado que sean 2XX (un 2 seguido de 2 números) indican que la carga ha ido bien. Por tanto, si el HTTP status de tu página web es 200 todo va como la seda.

El error 404 es un código de estado que se envía desde el servidor web al navegador, o sea, al usuario que intentaba entrar en dicha página. Básicamente, lo que indica este error es que se trata de un enlace roto, defectuoso o que ya no existe y que, por lo tanto, no es posible navegar por él. Si, es responsabilidad del desarrollador corregir

Como se mencionó anteriormente, el Error 500 - Internal server error es un error del lado del servidor , lo que significa que probablemente el problema no sea con tu ordenador o conexión a Internet sino con el servidor del sitio web. Es responsabilidad del desarrollador backend
